üìã Project Overview

This Logistic System is a complete implementation of a logistics management platform in Java. It handles shipment tracking, vehicle allocation, warehouse inventory management, and route optimization.

Key Classes:

Shipment Class

Manages individual shipments with tracking ID, status, and location information. Supports location updates and status changes.

Vehicle Class

Represents delivery vehicles with driver information and availability status. Can be assigned to shipments or released for new assignments.

Warehouse Class

Manages inventory using HashMap data structure. Supports adding and removing items with quantity tracking.

RouteOptimizer Class

Calculates and provides optimized routes between source and destination locations.
